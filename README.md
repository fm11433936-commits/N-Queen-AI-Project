#  N-Queens Solver using AI Algorithms

This project implements multiple **Artificial Intelligence algorithms** to solve the classic **N-Queens problem**, where the goal is to place N queens on an N×N chessboard without conflicts.

A detailed explanation of the project, algorithms, and results is included in the attached **PDF documentation**.

---

# Features 

* Solve N-Queens problem using multiple AI techniques:

  * Backtracking

  * Hill Climbing

  * Best-First Search

  * Genetic Algorithm

* Compare performance between different algorithms

* Visualize board states and solutions

* Modular and clean project structure
